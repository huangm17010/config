设置获取配置资源类型

设置获取配置资源类型（目前只支持默认的文件类型，可不配置），参数包括：

  • dataSourceType：配置资源类型，有效值包括：
             '0'：文件File
             '1'：关系型数据库Oracle
             '2'：NoSql数据库Cassandra

方法定义：

public ConfigCenter setDataSourceType(char dataSourceType){...};

示例代码：

ConfigCenter.getInstance().setDataSourceType('0');
