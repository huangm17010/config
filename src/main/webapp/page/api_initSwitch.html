初始化开关配置

返回首页

初始化开关配置，可指定并支持配置多个开关资源，建议在程序运行初期执行，参数包括：（ 设置类API需要在调用 initSwitch() 前执行）

  • sourceName：开关配置资源名，可不填，默认资源名"switch"

方法定义：

public ConfigCenter initSwitch(String... sourceName){...};

不指定开关资源名，默认加载"switch"资源，示例代码：

ConfigCenter.getInstance().initSwitch();

可指定多个开关配置资源名，如"switchhto","switch"等，示例代码：

ConfigCenter.getInstance().initSwitch("switchhto","switch");
