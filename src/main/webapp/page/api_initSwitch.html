初始化开关配置

初始化开关配置，在初始化配置中心 initConfig() 后执行，可指定并支持配置多个开关资源，建议在程序运行初期执行，参数包括：

  • sourceName：开关配置资源名，可不填，默认资源名"switch"

方法定义：

public ConfigCenter initSwitch(String... sourceName){...};

不指定开关资源名，默认加载"switch"资源，示例代码：

ConfigCenter.getInstance().initConfig().initSwitch();

可指定多个开关配置资源名，如"switchhto","switch"等，示例代码：

ConfigCenter.getInstance().setEnv("springbootdemo", "10.124.142.82", "8680").setScheduler(1000*60, 1000*60, false).initConfig().initSwitch("switchhto","switch");

注：建议在initConfig()后立即调用，如上示例
