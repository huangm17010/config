指定省分地市，获取对应服务名

返回首页

① 读取默认开关配置资源"switch"，传入省分地市编码、默认服务名和切换服务名，开关打开返回切换服务名，关闭返回默认服务名：

  • provinceCode：省分编码

  • eparchyCode：地市编码

  • defaultService：默认服务（开关关闭时返回）

  • switchService：切换服务（开关打开时返回）

方法定义：

public String switchService(String provinceCode,String eparchyCode,String defaultService,String switchService){...};

示例代码：

String serviceName=ConfigCenter.getInstance().switchService("11", "0010", "crm_userinfo", "crm_userinfojdbc");


② 读取指定开关配置资源名，其他同方法①：

  • sourceName：开关配置资源名

  • provinceCode：省分编码

  • eparchyCode：地市编码

  • defaultService：默认服务（开关关闭时返回）

  • switchService：切换服务（开关打开时返回）

方法定义：

public String switchService(String sourceName,String provinceCode,String eparchyCode,String defaultService,String switchService){...};

示例代码：

String serviceName=ConfigCenter.getInstance().switchService("switchhto", "11", "0010", "crm_userinfo", "crm_userinfojdbc");
