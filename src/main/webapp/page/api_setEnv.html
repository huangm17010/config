设置远端配置中心

返回首页

设置连接远端配置中心，建议只在本地开发时使用，平台应用通过环境变量获取这些参数，参数包括：

  • serviceName：服务名称

  • configIp：配置中心Ip地址

  • configPort：配置中心Port端口

方法定义：

public ConfigCenter setEnv(String serviceName,String configIp,String configPort){...};

示例代码：

ConfigCenter.getInstance().setEnv("springbootdemo","10.124.142.82","8680");

1.2.0版本，本地自动连接开发环境配置中心，如需读取服务特殊配置，只设置服务名即可

方法定义：

public ConfigCenter setEnv(String serviceName){...};

示例代码：

ConfigCenter.getInstance().setEnv("springbootdemo");
