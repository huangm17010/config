设置spring中，优先读取本地的配置项

返回首页

默认远端配置中心覆盖本地配置，如需使用本地配置，调用该方法，传入优先使用的本地配置项名称，参数包括：

  • propertyNames：配置项名称

方法定义：

public ConfigCenter useLocalSpringProperty(String... propertyNames){...};

示例代码：

ConfigCenter.getInstance().useLocalSpringProperty("spring.cache.type","server.port","server.tomcat.max-threads");
