指定省分地市，获取开关状态

返回首页

① 读取默认开关配置资源"switch"，传入省分地市编码，开关打开返回true，关闭返回false，参数包括：

  • provinceCode：省分编码

  • eparchyCode：地市编码

方法定义：

public boolean isOpenSwitch(String provinceCode,String eparchyCode){...};

示例代码：

boolean switchFlag = ConfigCenter.getInstance().isOpenSwitch("11", "0010");


② 读取指定开关配置资源名，其他同方法①，参数包括：：

  • sourceName：开关配置资源名

  • provinceCode：省分编码

  • eparchyCode：地市编码

方法定义：

public boolean isOpenSwitch(String sourceName,String provinceCode,String eparchyCode){...};

示例代码：

boolean switchFlag = ConfigCenter.getInstance().isOpenSwitch("switchhto","11", "0010");
