初始化配置中心，自动化加载配置项，建议在程序运行初期执行（初始化方法initConfig()只有第一次调用有效）

方法定义：

public ConfigCenter initConfig(){...};

无特殊设置需求，示例代码：

ConfigCenter.getInstance().initConfig();

有特殊需求，需要使用设置类API，示例代码：

ConfigCenter.getInstance().setScheduler(1000*60, 1000*60, false).setEnv("springbootdemo", "10.124.142.82", "8680").setDataSourceType('0').initConfig();

注：设置类API需要在initConfig()前调用

需要使用 1.1.0 版本新增开关功能，示例代码：

ConfigCenter.getInstance().setScheduler(1000*60, 1000*60, false).setEnv("springbootdemo", "10.124.142.82", "8680").setDataSourceType('0').initConfig().initSwitch();

注：initSwitch()方法需要在initConfig()后调用
