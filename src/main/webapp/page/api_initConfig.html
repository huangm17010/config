初始化配置中心

返回首页

初始化配置中心，自动化加载配置项到本地缓存，可在程序运行初期执行（只有第一次调用有效）

方法定义：

public ConfigCenter initConfig(){...};

无特殊设置需求，示例代码：

ConfigCenter.getInstance().initConfig();

需要使用设置类API，示例代码：( 设置类API需要在调用 initConfig() 前执行)

ConfigCenter.getInstance().setScheduler(1000*60, 1000*60, false).setEnv("springbootdemo").setDataSourceType('0').initConfig();
