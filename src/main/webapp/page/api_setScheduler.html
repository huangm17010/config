设置资源调度规则

返回首页

设置动态拉取配置项规则，参数包括：

  • initialDelayMillis：初始化配置成功后，首次动态加载配置时间间隔（单位毫秒，默认60000ms，即1分钟）

  • delayMillis：动态加载配置时间间隔（单位毫秒，默认60000ms，即1分钟）

  • ignoreDeletesFromSource：调度器是否忽略已经被删除的配置项（默认false，即程序运行期，保留所有加载过的配置项）

方法定义：

public ConfigCenter setScheduler(int initialDelayMillis,int delayMillis,boolean ignoreDeletesFromSource){...};

示例代码：

ConfigCenter.getInstance().setScheduler(1000*60, 1000*60, false);
